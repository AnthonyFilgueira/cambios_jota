# Tarea 2.3: Crear vista de historial

## Objetivo
Crear la vista `transactions/index.blade.php` que muestre el historial de transacciones del usuario.

## Componentes

### Widget de consumo acumulado
- Card destacada con total gastado
- Gradiente morado (cj-morado-profundo → cj-morado-medio)
- Icono de dinero
- Formato de moneda

### Tabla de transacciones
- Versión desktop (tabla completa)
- Versión mobile (cards apiladas)
- Responsive con Tailwind

### Badges de estado
- Pendiente: rosa (cj-rosa)
- En proceso: amarillo
- Completado: turquesa (cj-turquesa)
- Cancelado: gris

## Paleta Cambio J aplicada
- cj-morado-profundo
- cj-morado-medio
- cj-morado-claro
- cj-turquesa
- cj-rosa
- cj-texto
- cj-texto-claro

## Duración
10 minutos

## Prioridad
High
